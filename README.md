# 出張旅費申請書作成アプリ

会社PC・Mac・iPhone・iPadのブラウザから利用できる、端末内保存型の旅費申請Webアプリです。Webアプリ本体は `web/` にあります。

## Web版でできること

- 月単位の旅費行管理と同日複数行
- スクリーンショット、複数画像、PDF、CSV、ICS、テキスト、手入力から予定候補を作成
- OCR結果を確定前に確認・修正
- 予定を時刻順につないだ1日経路作成
- 自宅、複数出勤先、曜日別出発地・戻り先、訪問先、定期券の登録
- 定期券内0円経路を内部保持し、通常一覧と出力から除外
- 確認、修正、保留、除外、申請済みと重複候補警告
- IndexedDBへの端末内保存とJSONバックアップ・復元
- 20行単位のExcel貼り付け用コピーと列別コピー
- 2026年度版XLSX原本を使ったExcel申請書作成

画像、PDF、予定情報を外部AIサービスへ送信しません。OCRはブラウザ内で実行します。公開サーバーには利用者の旅費データを保存しません。

## 構成

- `web/app/`：React/TypeScript画面と業務処理
- `web/app/lib/db.ts`：IndexedDB保存
- `web/app/lib/domain.ts`：経路、定期券、出力対象、コピー処理
- `web/app/lib/excel.ts`：原本XLSXへの書き込み
- `web/tests/`：経路・分割・定期券・描画テスト
- `src/travel_expense/`：従来のPython版Excel出力エンジン（削除せず保存）

## 原本について

Excel自動作成では「2026年度版出張旅費代精算書原本」の `.xlsx` を利用者が画面で選択します。原本は読み取り専用で扱い、新しいファイルとして保存します。原本を選択できない場合や自動作成に失敗した場合も、コピー出力を利用できます。

## 従来のPython版

2026年度版「出張旅費代精算書」原本へ、確定済みの旅費明細を出力するPythonエンジンです。

## 実装済み

- XLSX原本の書式・結合セル・数式を保持した別名出力
- ODS原本の作業用コピーをLibreOfficeでXLSXへ変換（原本は変更しない）
- 対象月・確認状態・金額・除外・申請済み・重複警告による抽出
- 日付、開始時刻、経路順、作成順による並べ替え
- 20行単位の複数シート分割、枚数表示、ページ小計・総合計照合
- 電車・バス内訳、定期変更、ヘッダー、提出日の出力
- 原本構造と保存後ファイルの再読込検証
- 同名ファイルの無断上書き防止

## セットアップと実行

```powershell
py -m venv .venv
.venv\Scripts\pip install -e .
travel-expense-export "原本.ods" examples\request.json --output "精算書.xlsx"
```

ODS入力にはLibreOfficeの `soffice` がPATH上に必要です。JSON構造は `examples/request.json` を参照してください。成功時は、保存先・明細数・用紙数・合計をJSONで標準出力します。

出力成功だけでは元データを申請済みに変更しません。呼び出し側は利用者が出力内容を確認した後に、返却された明細IDを使って申請済み履歴を確定してください。

## テスト

```powershell
.venv\Scripts\pip install -e .
py -m unittest discover -v
```

原本未添付のため、テストでは同じセル構造を持つ最小原本を生成しています。実原本での最終確認では、罫線・行高・列幅・印刷結果をExcel上でも確認してください。
